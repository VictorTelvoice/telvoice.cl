Telvoice.cl · Chile

API SMS Chile para desarrolladores

Conecta tu software, CRM o automatización con la API REST de Telvoice para enviar SMS transaccionales y campañas desde Chile hacia móviles locales.

Capacidades de la API

  • Envío unitario y masivo vía HTTPS JSON.
  • Ambiente sandbox para pruebas sin impacto en producción.
  • Webhooks de entrega (DLR) y trazabilidad por message ID.
  • Rate limits configurables y scopes por API key.

Casos de uso

Notificaciones de pedidos, OTP, alertas de seguridad, recordatorios de pago, integraciones con n8n, Zapier, backends Node/Python y apps móviles.

Preguntas frecuentes

¿Cómo obtengo una API key?
Tras comprar una bolsa SMS y activar tu cuenta, solicita acceso API desde el panel o contacto comercial.
¿Hay ambiente de prueba?
Sí, Telvoice ofrece keys sandbox para desarrollo antes de pasar a producción.
¿Documentación disponible?
Documentación PDF/HTML desde el panel cliente y guías en el centro de ayuda.
