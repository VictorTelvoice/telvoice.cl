Cómo funciona

Operación SMS con asistencia guiada

Prepara tu campaña, valida tu base, revisa el consumo estimado y confirma antes de enviar. El agente te guía paso a paso para reducir errores y mantener control sobre cada envío.

  1. 1

    Define tu mensaje

    Indica si enviarás una promoción, alerta, recordatorio, cobranza o notificación a tus clientes.

  2. 2

    Prepara destinatarios

    Carga un CSV o usa una plantilla guardada. El agente revisa formato, duplicados y números válidos.

  3. 3

    Revisa consumo y saldo

    Visualiza SMS estimados, segmentos, costo y saldo requerido antes de ejecutar.

  4. 4

    Confirma y mide

    Envía solo con confirmación y revisa estados de entrega, DLR y resultados desde el panel.

Preguntas frecuentes

¿Tienes dudas?

Revisa las guías del centro de ayuda para el portal SMS o consulta las preguntas más comunes aquí.

¿Qué es una bolsa de SMS? add

Es una cantidad prepagada de mensajes que puedes comprar para enviar campañas, notificaciones o recordatorios a teléfonos en Chile.

¿Cómo funciona el envío de SMS masivos? add

Puedes enviar SMS desde tu sistema vía nuestra API REST o desde el panel de administración web. Solo debes subir tu base de datos de teléfonos, redactar el mensaje y programar el envío. Los mensajes se procesan hacia la red chilena con rutas orientadas a una entrega estable.

¿En cuánto tiempo se integra la API? add

La integración vía API dependerá del sistema del cliente, el tipo de uso y la validación técnica. Telvoice puede entregar orientación para facilitar el proceso, junto con documentación y ejemplos según el tipo de integración requerida.

¿Qué operadores móviles incluyen? add

Cubrimos los cuatro operadores principales de Chile: Entel, Movistar, Claro y WOM, con rutas orientadas a maximizar la entrega hacia sus redes móviles.

¿Los SMS vencen? add

Las condiciones de uso y vigencia se informan al momento de comprar cada bolsa. Para volúmenes altos podemos acordar condiciones especiales en tu cotización.

¿Tienen soporte técnico en Chile? add

Sí. Telvoice.cl es operado por TelefoníaChile LTDA, empresa establecida en Chile. Nuestro equipo técnico y comercial está disponible en horario local y puedes comunicarte en español para cualquier consulta o incidencia.

¿Cómo se emite la factura? add

Emitimos factura electrónica afecta al IVA por cada período facturado, en pesos chilenos (CLP). La factura se envía automáticamente al correo registrado y queda disponible en el portal del SII bajo el RUT de TelefoníaChile LTDA.

¿Puedo enviar SMS desde mi propio sistema? add

Sí. Telvoice puede entregar integración vía API para empresas que necesitan automatizar envíos.

¿Sirve para campañas promocionales? add

Sí, siempre que la empresa utilice bases autorizadas y respete las buenas prácticas de comunicación.

¿Puedo cotizar grandes volúmenes? add

Sí. Para volúmenes sobre 100.000 SMS mensuales puedes solicitar una cotización especial desde la sección Empresas o el formulario de contacto.

¿Telvoice.cl es lo mismo que portal.telvoice.net? add

Telvoice.cl está enfocado en el mercado chileno y en la compra de bolsas SMS. portal.telvoice.net es el portal cliente donde envías mensajes, revisas reportes y administras tu cuenta.

¿El agente envía SMS automáticamente? add

No. El agente puede preparar, calcular y resumir una acción, pero los envíos y débitos requieren confirmación explícita.

¿Puedo empezar sin integrar API? add

Sí. Puedes operar desde el panel comprando saldo, cargando contactos, preparando campañas y revisando resultados.

¿Telvoice ofrece numeración propia? add

Sí. Telvoice puede habilitar numeración propia, números dedicados y canal SMS empresarial para OTP, bots, API/webhooks y agentes conectados a tu operación, según validación comercial y configuración técnica.

¿Puedo usarlo para OTP o validaciones? add

Sí, bajo configuración técnica y validación comercial según el tipo de tráfico, volumen y caso de uso.

¿La comunicación es completamente autónoma? add

La plataforma puede asistir flujos operativos, pero las acciones sensibles deben operar con reglas, permisos y confirmación explícita.
